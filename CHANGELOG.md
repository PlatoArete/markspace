## [0.1.0] - 2025-XX-XX

### Added
- Initial release
- Markdown editing with CodeMirror 6
- Hybrid live preview (rendered headers, lists, links)
- Workspace sidebar with file tree
- Tab-based file management with drag-and-drop reordering
- Hot exit (unsaved changes persist across sessions)
- Quick Links for pinning favorite folders
- Global search across workspace
- Find & replace with regex support
- Command palette
- Copy as HTML
- Export to PDF
- Export to DOCX
- Light, Dark, and Hybrid themes
- Vim mode (optional)
- Customisable font size, line numbers, word wrap, accent colour
